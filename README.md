# AI 感情助手

## 项目介绍

AI 感情助手是一个基于 Streamlit 和 OpenAI API 构建的智能聊天应用，支持自定义 AI 角色名称、性格描述，可创建 / 切换 / 删除多个聊天会话，并且所有会话记录会持久化存储在本地，同时提供流式消息交互体验，模拟真实的对话场景。

## 功能特性

- 🗨️ 实时流式聊天：用户消息和 AI 回复均采用流式输出，模拟打字效果
- 💬 多会话管理：支持新建、切换、删除聊天会话，会话数据本地持久化
- 🎭 自定义 AI 角色：可修改 AI 名称、自定义性格描述，打造专属聊天助手
- 📝 会话记录保存：所有聊天记录自动保存至本地文件，重启应用不丢失
- 🚀 流式响应：AI 回复采用流式输出，提升交互体验

## 环境要求

- Python 3.8+
- 有效的 OpenAI API Key（或兼容 OpenAI 接口的第三方大模型 API Key）

## 安装步骤

### 1. 克隆项目
```bash
git clone <你的项目仓库地址>
cd <项目文件夹>
```
### 2. 安装依赖


``` bash
uv sync
```
如果没有下载uv记得下载
```bash
pip install uv
```
### 3. 配置环境变量
```
# OpenAI API 基础地址（支持第三方兼容接口，如本地部署的模型、阿里云百炼等）
BASE_URL=https://api.openai.com/v1
# 你的API Key
API_KEY=your_openai_api_key_here
# 使用的模型名称（如gpt-3.5-turbo、gpt-4等）
MODULE=gpt-3.5-turbo
```

## 使用说明

### 启动应用
```bash
streamlit run aichat.py
```

### 核心操作

1. **基础聊天**：在底部输入框输入消息，回车发送，AI 会以流式方式回复
2. **会话管理**（侧边栏）：
    - 新建会话：点击「新建会话」按钮，清空当前聊天记录并创建新会话
    - 切换会话：点击侧边栏展示的历史会话名称，即可切换至对应会话
    - 删除会话：点击会话右侧的「删除」按钮，可删除该会话及记录
3. **AI 角色配置**（侧边栏）：
    - 人物名称：输入自定义的 AI 名称（如 "小萌"、"客服助手"）
    - 自定义人物描述：输入详细的性格 / 行为描述（如 "温柔的情感顾问，擅长倾听和安慰"）

## 数据存储说明

- 所有会话数据存储在项目根目录的`data`文件夹中，每个会话对应一个以时间戳命名的 JSON 文件
- JSON 文件包含以下信息：
    
    - `AIChatName`：AI 角色名称
    - `AIChatTrait`：AI 角色性格描述
    - `messages`：聊天记录（包含用户和 AI 的消息）
    - `Key`：会话唯一标识（时间戳）
    

## 注意事项

1. 确保`.env`文件中的 API Key 和 BASE_URL 配置正确，否则无法正常调用 AI 接口
2. 若使用第三方兼容 OpenAI 接口的模型，需确认`MODULE`配置与模型名称匹配
3. 会话删除操作不可逆，请谨慎操作
4. 本地存储的`data`文件夹建议定期备份，避免聊天记录丢失

## 常见问题

1. **启动后提示 API Key 错误**：检查`.env`文件中`API_KEY`是否正确，或确认网络可访问配置的`BASE_URL`
2. **流式输出不生效**：确保 OpenAI API 版本≥1.0.0，且模型支持流式响应
3. **切换会话后记录未更新**：切换会话后应用会自动`rerun`，若未生效可刷新页面
4. **删除会话无反应**：确认文件权限，确保应用有权限删除`data`目录下的文件

## 技术栈

- 前端框架：Streamlit（快速构建 Web 应用）
- 后端交互：OpenAI Python SDK（调用大模型 API）
- 数据存储：JSON 文件本地持久化
- 环境管理：python-dotenv（加载环境变量）
